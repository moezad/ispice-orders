Setup Required — tap to see how to connect orders across devices (one-time, 5 min)
1

Open sheets.google.com and create a new spreadsheet. Name it iSpice Orders. Then go to Extensions → Apps Script.

2

Delete all existing code in Apps Script. Then open the file apps-script.gs that came with this package, copy all of it, and paste it into Apps Script. Click Save (💾).

3

Click Deploy → New deployment. Set type to Web app. Set "Execute as" to Me and "Who has access" to Anyone. Click Deploy and copy the URL it gives you.

4

Open both index.html and dashboard.html in a text editor. Find the line const SCRIPT_URL = ''; near the top of each file, and paste your URL between the quotes. Save both files and re-upload to GitHub.

5

Done! Orders from any device will now appear here instantly. Your Google Sheet also acts as a live record — the rep can open it directly anytime.
